# SmartPN Atlas Website

Official static website source for SmartPN Atlas.

## Purpose

This repository supports the LinkedIn + website publishing pipeline:

- Jim confirms a LinkedIn post and image.
- The short post is published to LinkedIn.
- The fuller article is published under `/insights/` on smartpn.com.tw.
- Netlify deploys the website from the `main` branch.

## Netlify Settings

- Build command: none
- Publish directory: `.`
- Production domain: `smartpn.com.tw`
- Continuous deployment source: `nethinetkimo01-afk/smartpn-site`, branch `main`

## Product Positioning

SmartPN Atlas is a manufacturing-side supply chain material identity governance layer. It helps supplier, OEM, and brand systems reference the same material identity without replacing ERP, PDM, PLM, or existing supplier systems.
